# Mobile Phone Recommendation System

## Overview

This project implements recommendation system techniques using mobile phone user review data. The objective is to recommend products to users based on historical ratings and review patterns.

The project explores both popularity-based and collaborative filtering approaches to understand different recommendation strategies and their effectiveness.

## Domain

E-Commerce / Recommendation Systems

## Problem Statement

With thousands of products available online, recommendation systems help users discover products that match their interests and preferences.

This project uses customer review and rating data to build recommendation models capable of suggesting relevant mobile phones to users.
